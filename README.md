# ✈️ Sistema de Passagens Aéreas

Projeto desenvolvido em Java com foco na aplicação de Programação Orientada a Objetos (POO).

## 📌 Funcionalidades

* Criação de passagens aéreas
* Tipos de passagem:

  * Econômica (+10%)
  * Executiva (+25%)
* Cálculo automático do preço final
* Associação de cliente com a compra
* Simulação de venda

## 🧠 Conceitos aplicados

* Herança
* Polimorfismo
* Abstração
* Encapsulamento
* Composição

## 💻 Tecnologias utilizadas

* Java

## ▶️ Como executar

1. Abra o projeto em uma IDE (IntelliJ, Eclipse, etc.)
2. Execute a classe `Main.java`
